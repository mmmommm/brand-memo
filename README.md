# brandmemo  
  
## 簡単にいうと  
  
自分で調べた企業の情報について簡潔にメモしておける物があればいいと思い作成したサイトです（まだ途中ですが、、、）  
現段階では私の作成したメモが見られるだけなので、最低限のページネーション、検索機能が出来上がったらデプロイする予定です。  
最終的にはログイン機能をつけ、それぞれログインしていただいた方が自分でメモをつくれるようになることです。
